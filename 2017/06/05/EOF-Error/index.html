EOF Error

2017.06.05 Seg Xiong errors

今天队友上传了 JDStore 大赛的第一版作品，测试的时候点击生成订单出现如下图的报错

由于不是使用的自己的heroku账号，无法查看 heroku logs，所以拉下本地进行测试。在 git pull 之后重开 server，进行到同样的步骤（生成订单）的时候，出现如下图的报错信息。

在一番Google之后将错误的可能锁定在生成订单的同时向用户发送邮件的 action mailer，在controller中注释掉 Mailer 之后再测试，报错问题消失。

由于我们使用的是 sendcloud 邮件服务，所以在专门针对 sendcloud 进行问题搜索，最后的发现了类似的案例。

最后联络队友，查询 sendcloud 账户设置，最终找到问题根源。

最后想了解更多 EOFError 的，可自行参考此链接。
