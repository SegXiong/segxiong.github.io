Capybara 'fill_in'

2017.09.19 Seg Xiong Tips

最近用Capybara做用户测试的时候总是遇到 Capybara::ElementNotFound: 错误

仔细检查代码，发现原来是错把 <label> 的值放到 fill_in 造成的

一时好奇查阅了 capybara 的文档得知原来 fill_in 除了通过 label text 定位以外还能使用 id 和 name 捕获。

Locate a text field or text area and fill it in with the given text The field can be found via its name, id or label text.
