CSS基础知识

2017.04.14 Seg Xiong Tips

  • margin 与 padding 的差异
    字面上来说，margin是外边距而padding是内边距。直接包围内容的是内边距，内边距呈现了元素的背景，内边距的边缘是边框。边框以外是外边距，外边距默认是透明的。另外内边距不接受负值，而外边距可以为负。

  • 什么是 box model
    即框模型，规定了元素框处理元素内容、内边距、边框和外边距的方式。框模型由里到外分别是元素(element)，内边距(padding)，边框(border)和外边距(margin)。外边距默认是透明的，因此不会遮挡其后的任何元素。内边距、边框和外边距都是可选的，默认值是零。在 CSS 中，width 和 height 指的是内容区域的宽度和高度。增加内边距、边框和外边距不会影响内容区域的尺寸，但是会增加元素框的总尺寸。

  • 为何要使用 em 而非 px 来定义字的大小
    设置字体大小有两种方式，px(绝对值)和em(相对值)。1em等于当前的字体尺寸，浏览器的默认字体是16px，1em等于16px。除IE浏览器外，用户能使用绝大多主流数浏览器对字体进行重新调整。为了避免在 Internet Explorer 中无法调整文本的问题，许多开发者使用em单位代替px。在所有浏览器中调整字体均有效的方案是为\元素(父元素)以百分比设置默认的font-size值。

  • h1 {margin : 10px 0px 15px 5px;}
    这是设置margin属性的一种简写方式，其属性值按顺时针分别表示margin-top(10px)，margin-right(0px)，margin-bottom(15px)，margin-left(15px)。
