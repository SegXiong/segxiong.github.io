HTML相关的一些基础概念

2017.04.13 Seg Xiong Tips

以下是HTML里有一些新手容易搞混的概念的区别：

​

  • <div> 与 <span>
    字面上的区别是<div>是块级元素，在显示此元素的前后浏览器会自动换行。<span>是行内元素，在显示此元素的前后浏览器不会自动换行。前者多用于对大块的内容设置样式属性，后者用于对大块内容中的部分内容设置样式属性。
    ​
  • class 与 id
    两者都是元素属性，字面上的区别是class是对元素的类别进行设置，id则是对个体的身份进行设置。class与id的主要在于唯一性，id是唯一的，同一个id只能给某一个元素，而同一个class则可以分配给若干个元素。
    ​
  • <p> 与 <br>
    从类型上看<p>属于块级元素，<br>属于空元素。虽然显示上两者都会换行，但是<p>元素内可以填充内容或其他元素，而<br>则不行。
    ​

    使用 table 排版

    ​
    使用 table 排版主要需要搞清楚几个 table 内使用的标签的含义：
    ​
  • <table>
    <table>标签定义了整个表格的范围，以下的标签全部都只能出现在此标签的内部。
    ​
  • <caption>
    <caption>标签定义了表格的标题，一般紧跟在<table>标签之后。
    ​
  • <tr>
    <tr>标签定义了表格的行。一对<tr>``</tr>标签就是一行表格。
    ​
  • <th>
    <th>标签定义了表头。一般情况下出现在表格的第一行或者最后一行內。内容会加粗显示。
    ​
  • <td>
    <td>标签定义了表格单元。如果单元内容为空应尽量用<td>&nbsp;</td>代替<td></td>。
    ​
  • <thead> <tbody> <tfoot>
    这个三个标签分别定义了表格的也没页眉，主体，页脚。用于表格的样式设置，一般嵌套于<tr>标签外部。
    ​
