meetup 小结(5.7)

2017.05.07 Seg Xiong meetup

本周是Job Listing魔改大赛的最后一周，几乎每天都时间都花在了自己作品的页面改善上。
上线了第二版内容，包括页面美化，首页图片轮播，动态toolbar和收藏功能。
由于本周大家都专注于为自己网站做最后的冲刺，以至于这从周末的分享几乎都集中在前端和大赛心得。以下是部分摘录总结。

  1. 摆正心态。参加本次大赛的同学来自五湖四海，各行各业，有的同学本身就有作为程序员的经验，所以大家的起点不同也是完全可以理解的，不需要和别的同学做比较。应尽量和昨天的自己做比较，每天做了什么，对比昨天学到了什么，这样的比较才有正向的激励作用。

  2. 不要忽视营销。有的同学因为自己的作品比较简陋就羞于在群里宣传和为自己的作品拉票，其实这是一种自负的表现，好的产品都是通过不断的迭代，以不断的趋于成熟。有这种想法的同学应该尽早修正自己的观念。好的营销和好的作品是不冲突的，好的产品也需要营销。

  3. CSS Diner。一个富有趣味性的CSS选择器教程，动手性很强的小工具。

  4. Downie。是一款Mac上的网页视频抓取软件，可以实时下载，非常实用。
