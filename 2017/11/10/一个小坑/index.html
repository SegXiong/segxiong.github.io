一个小坑

2017.11.10 Seg Xiong Tips

今天新建了一个 Rails 项目，在套版 bootstrap 通知 flash 的时候发现关闭按钮不生效了。

第一反应是检查 application.js 的 require_tree

下面是以前的项目的 require_tree

果然没有 require jQuery ， 检查使用的 Rails 版本后发现已经更新到 5.1.4 ，查阅 5.1 版本的 release note 后发现官方已经移除 jQuery 的默认引用。rails_ujs 也由纯 js 重写，移除了对 jQuery 的依赖。

如果要继续使用 bootstrap 就必须手动载入 jQuery。

具体步骤依次执行如下指令(以装有homebrew的 Mac 为例)，

brew install yarn

yarn add jquery

在 application.js 中 //= require jquery

刷新页面再试试，问题解决。
